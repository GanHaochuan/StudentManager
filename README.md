# 作业提交说明

## 主项目：StudentManager（分支 master）
一个基于C++面向对象编程的学生成绩管理系统，实现了学生信息管理、成绩计算与展示功能。项目采用模块化设计，使用CMake构建。

## 额外项目：CS106B Assignments（分支 extra）
包含CS106B课程的七个编程作业，涵盖递归、数据结构、图算法、堆、哈希表等核心主题，展示了在算法与数据结构方面的学习成果。

两个项目均已上传至GitHub仓库，详细的项目介绍、功能说明和运行方式请查看各自目录中的README文件。
